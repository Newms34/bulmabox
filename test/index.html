Generic (Default) Dialogs
Button Syntax* Additonal Info
bulmabox.alert('Some Title','Some Message',callbackFn(),options) A regular old alert button. Can accept a callback, but does not need one!
bulmabox.confirm('Some Title','Some Message',callbackFn(),options) A confirm dialog, providing continue ("Ok") or stop ("Cancel") options. Must have a callback (or Bulmabox will throw an error).
bulmabox.prompt('Some Title','Some Message',callbackFn(),options) A prompt, for telling us something. Must have a callback (or Bulmabox will throw an error).
bulmabox.custom('Some Title','Some Message',callbackFn(),options) Fancy custom dialog! You can put anything in here! Must have a callback (or Bulmabox will throw an error).
*Italicized parameters are required.
Custom Language and Butttons
Button Syntax* Additonal Info
bulmabox.alert('Some Title','Some Message',callbackFn(),{lang:'language code'}) By passing {lang:'la'} as the final option to alerts, confirms, propmts, and custom dialogs, we can set our button language.
bulmabox.confirm('Some Title','Some Message',callbackFn(),{okay:{text:'Indeed!',colors:{fg:'blue',bg:'green'}}}) Or we can just completely override the button colors and text!
*Italicized parameters are required.
Global Configuration
Button Syntax Additonal Info
bulmabox.opts({globalAlert:true}) Override the default window.alert() method.
bulmabox.opts({lang:'fr'}) Set Global Language to 'Oui' and 'Non' (French). Note that French buttons normally are "OK" and "Annuler". I'm doing this so that you can see the global overrides!
Also, the language must be in my list of languages.
bulmabox.opts({hfBg:'#bbf'}) We can set the global background for the dialogs' header and footer.
bulmabox.opts({mainBg:'#bfb'}) We can also set the main background. Note that for this (and the header/footer backgrounds), we're simply passing whatever value you give it to the CSS background property, so you can use images if you want!
bulmabox.opts({reset:true}) Finally, we can just reset everything. Note that this overrides everything else except the global alerts!
